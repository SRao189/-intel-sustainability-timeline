# Reflection Questions

## 1. What was challenging, and how did you use AI?

- Which layout or interaction decision took the most iteration?
- What did AI help you do faster: brainstorming, refining CSS, or debugging responsive behavior?
- What did you still have to check manually after using AI?
- What surprised you about how the design changed from the first draft to the final version?

## 2. How would you discuss this project while networking?

- Explain the project in one sentence first.
- Mention the responsive timeline layout and why it matters.
- Point out one accessibility decision and one visual-design decision.
- Describe how you used Git, testing, and iteration to finish the assignment cleanly.

## 3. What about Intel's culture interested you, and how does it connect to your goals?

- Which themes stood out most: sustainability, engineering discipline, long-term planning, or global impact?
- How does that connect to the kind of work you want to do?
- What part of the project made you more interested in technology, design, or responsible innovation?
